# Screenshots

This directory contains screenshots captured during each release.

## Purpose

Screenshots provide visual verification that the platform is functioning as expected.

## Recommended Screenshots

### Repository

* GitHub repository homepage
* Repository tree

### Platform

* Bootstrap output
* Doctor output
* Verify output
* Docker containers (`docker ps`)
* HAProxy dashboard

### Applications

* Frontend
* Admin
* API

### TLS

* Browser certificate information
* HTTPS verification

## Naming Convention

Use release-based names.

Example:

```
v0.1.0-bootstrap.png
v0.1.0-api.png
v0.1.0-haproxy-dashboard.png
```

Future releases should maintain the same naming convention.
