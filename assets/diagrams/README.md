# Architecture Diagrams

This directory contains exported architecture diagrams used throughout the repository.

## Purpose

Diagrams provide a visual representation of the platform architecture at each release.

## Source Files

Editable Draw.io diagrams are stored in

```
assets/drawio/
```

Exported images (PNG/SVG) should be stored here.

## Naming Convention

```
architecture-v0.1.0.png

architecture-v0.2.0.png

...
```

## Update Policy

Whenever the platform architecture changes significantly, update the Draw.io source first and then export the updated diagram into this directory.
