# Code of Conduct

This project is intended to be a welcoming place for engineers who want to learn Platform Engineering, DevOps, Cloud Infrastructure, and Production Operations.

## Our Standards

Participants are expected to:

* Be respectful.
* Provide constructive feedback.
* Share knowledge openly.
* Help others learn.
* Keep discussions professional.

## Unacceptable Behavior

Examples include:

* Harassment
* Personal attacks
* Discrimination
* Offensive language
* Deliberate misinformation

## Enforcement

Project maintainers may remove content or contributions that violate this code of conduct.

The goal is to maintain a positive and collaborative learning environment.
