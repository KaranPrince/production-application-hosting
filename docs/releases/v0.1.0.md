# Release v0.1.0

## Release Name

Production Edge

---

## Summary

The first production-ready platform edge has been implemented.

This release introduces secure multi-service hosting behind a single reverse proxy with HTTPS, internal PKI, health monitoring, and operational tooling.

---

## Features

* Docker Compose deployment.
* HAProxy reverse proxy.
* Host-based routing.
* Internal Root Certificate Authority.
* TLS termination.
* Docker DNS.
* Flask API.
* Backend health checks.
* HAProxy statistics dashboard.
* Packet capture.
* Production incident simulations.

---

## Known Limitations

* Self-signed certificates.
* Flask development server.
* Static frontend.
* Single HAProxy instance.
* Manual certificate generation.
* No persistent database.
