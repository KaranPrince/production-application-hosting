# Release v0.1.0 — Foundation

Release Date: 2026-07-06

## Overview

This is the initial public release of the **Production Application Hosting** learning repository.

The focus of this release is establishing a reproducible local platform that introduces the core building blocks of modern application hosting.

## Included

* Repository foundation
* Docker Compose platform
* HAProxy reverse proxy
* HTTPS with a custom Root CA
* Flask API service
* Frontend service
* Admin service
* PKI automation
* Operational scripts
* Sprint-00 documentation
* Sprint-01 documentation
* Initial runbooks
* Architecture Decision Records

## Verification

The release has been validated using:

* Bootstrap automation
* Platform verification
* HTTPS validation
* HAProxy routing
* Certificate generation and rotation

## Known Limitations

This release intentionally does not include:

* Public DNS
* Databases
* Kubernetes
* CI/CD
* Monitoring
* Logging
* Service discovery
* Cloud infrastructure

These topics are introduced progressively in future releases.

## Next Release

**v0.2.0 — DNS Fundamentals**

The next release introduces name resolution, DNS records, recursive resolution, authoritative DNS, and production DNS architecture.
